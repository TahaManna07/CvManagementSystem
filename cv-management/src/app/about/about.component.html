À propos de CV Management System

Le CV Management System est une plateforme avancée conçue pour optimiser le processus de gestion, d'analyse et de catégorisation des CVs. Construite avec des technologies de pointe, cette solution assure une intégration fluide entre le frontend, le backend et les composants LLM pour offrir un outil puissant de gestion des CVs.

Comment ça fonctionne

Le CV Management System traite les CVs téléchargés en extrayant des informations essentielles telles que les coordonnées, les compétences et les profils de poste adaptés. Les données extraites sont ensuite stockées dans une base de données MySQL et catégorisées à l'aide de techniques avancées d'LLM. La plateforme offre une interface intuitive pour visualiser et gérer les CVs, ainsi que des analyses approfondies pour aider dans les processus de recrutement.

Technologies utilisées

Frontend

Angular : Un framework polyvalent pour développer une interface utilisateur dynamique et réactive.
Angular Material : Fournit un design UI cohérent et intuitif avec une collection de composants réutilisables.

Backend

Python & Flask : Python est utilisé pour la logique backend, avec Flask servant de framework léger pour créer des APIs RESTful.
MySQL : Gère et stocke efficacement les données structurées des CVs.
Google AI Gemini : Utilisé pour générer et analyser du contenu, offrant des informations approfondies sur les données traitées par le système.

Déploiement

Docker : La containerisation assure un déploiement fluide et une évolutivité de l'application dans différents environnements.
Docker Compose : Gère les applications Docker multi-conteneurs pour les services frontend et backend.
